# SailMaster

SailMaster raises and lowers Sailwind sails by type.

Default hotkeys:

- `0`: all sails
- `1`: group 1
- `2`: group 2
- `3`: group 3
- `4`: group 4
- `5`: group 5
- `6`: group 6
- `F7`: show or hide the SailMaster panel

The mod decides whether a group should raise or lower from the current rope state. If any sail in the selected group is fully or partially deployed, the group lowers. If every sail in the selected group is furled, the group raises.

The SailMaster panel detects controllable sails on the current ship. Each sail has a slider for its deployed amount plus Min and Max buttons. Six runtime groups are available in the panel; select a group, check sails into it, then use the group controls or number hotkeys to set those sails together.
