# SailMaster

SailMaster raises and lowers Sailwind sails by type.

Default hotkeys:

- `0`: all sails
- `1`: square sails
- `2`: lateen sails
- `3`: junk sails
- `4`: gaff sails
- `5`: staysails
- `6`: other sails, including fin sails for now

The mod decides whether a group should raise or lower from the current rope state. If any sail in the selected group is fully or partially deployed, the group lowers. If every sail in the selected group is furled, the group raises.
